unpersist {SparkR}R Documentation

Unpersist an RDD

Description

Mark the RDD as non-persistent, and remove all blocks for it from memory and disk.

Usage

unpersist(rdd)

## S4 method for signature 'RDD'
unpersist(rdd)

Arguments

rdd

The RDD to unpersist

Examples

##D sc <- sparkR.init()
##D rdd <- parallelize(sc, 1:10, 2L)
##D cache(rdd) # rdd@env$isCached == TRUE
##D unpersist(rdd) # rdd@env$isCached == FALSE
## End(Not run)
